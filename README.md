## MC Wiki Crawler

1. `main.py`
   - `crawl`: crawl data
   - `split_content`: split files whose content maybe too long as input to gpts
2. `crawler_data`
   - `Block`, `Item`, `Mob`: not split files data
   - `_split` suffix: split files data
